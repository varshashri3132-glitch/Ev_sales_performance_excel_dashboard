
# EV Sales Performance Dashboard (Excel)

## Project Overview

This project presents an interactive Excel dashboard that analyzes electric vehicle (EV) sales performance across different regions, brands, vehicle types, and customer segments.

The goal of this project is to transform raw EV sales data into meaningful business insights using Excel’s data analysis and visualization capabilities. The dashboard allows users to explore sales trends, identify top-performing brands and models, and understand customer segment distribution.

---

## Dataset

The dataset includes electric vehicle sales information with the following fields:

* Date
* Region
* Brand
* Model
* Vehicle Type
* Battery Capacity
* Customer Segment
* Fast Charging Option
* Units Sold
* Revenue

The dataset enables analysis of EV market performance across multiple business dimensions.

## Dashboard KPIs

The dashboard highlights key performance indicators including:

* Total Units Sold
* Total Revenue
* Top EV Brand
* Top EV Model
* Top Vehicle Type

These KPIs provide a quick overview of overall EV market performance.

---

## Dashboard Visualizations

### EV Sales Based on Battery Capacity

Shows how EV sales vary across different battery capacity ranges, helping understand consumer preference for vehicle range.

### Fast Charging Capability Distribution

Displays the percentage of EVs equipped with fast charging versus those without fast charging capability.

### EV Sales by Vehicle Type

Breakdown of EV sales by vehicle category such as:

* Crossover
* Sedan
* Coupe
* SUV
* Truck
* Hatchback

### Customer Segment Distribution

Analyzes EV adoption across different customer groups such as:

* Budget Conscious
* High Income
* Middle Income

### Monthly EV Sales Trend

A time series visualization showing how EV unit sales change throughout the year.

---

## Key Insights

* Crossovers and hatchbacks account for the highest EV sales.
* A majority of EV models include fast-charging capability.
* Battery capacity between mid-to-high ranges shows strong adoption.
* Certain brands dominate EV sales performance.
* Sales trends fluctuate across months indicating seasonal demand patterns.

---

## How to Use the Dashboard

1. Download the Excel file from this repository.
2. Open the file using Microsoft Excel.
3. Use the **Region slicer** to filter sales by geographic location.
4. Use the **Brand slicer** to analyze specific EV manufacturers.
5. Interact with charts to explore EV market insights.
